Meus Projetos

Análise de Dados State of Data Brasil

O "State of Data Brazil 2023", é uma base de dados que busca reunir informações dos profissionais da área de dados e disponibilizá-las para que a comunidade desfrute e crie análises desses dados conforme desejar. Inspirada por isso, a comunidade Data Girls escolheu essa base de dados como desafio final da segunda edição do seu bootcamp Recomeçar. As participantes desse bootcamp foram divididas em grupos de quatro pessoas para realizarem essa análise final, que juntas devem explorar os dados e trazer insights, utilizando os conhecimentos adquiridos ao longo do bootcamp.

Clusterização dos dados do Spotify

O projeto consistiu na criação de um software de recomendações musicais baseado em uma base de dados clusterizada, com o objetivo de ajudar usuários a descobrir novos artistas em seus gêneros favoritos. Utilizando o algoritmo KMeans do sklearn, foram criados 80 clusters a partir de 112 gêneros musicais, número escolhido por reduzir a complexidade dos dados e melhorar a identificação de padrões. Esse agrupamento facilitou a segmentação de grandes volumes de dados e possibilitou a personalização eficiente das recomendações, com base em uma base de dados do Kaggle.
